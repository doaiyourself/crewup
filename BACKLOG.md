# Crew Up — 향후 구현 백로그 (나중에 할 일)

> 출시 후 사용자 리텐션이 확보되면 진행할 기능. 지금은 **핵심 도구(출퇴근·할일·급여·계약서)**에 집중.

---

## 1. 💰 광고 수익화 (애드센스 / 애드몹)

**한 줄 요약:** 무료 사용자에게만 광고 노출 → 광고 자체를 "업그레이드 압박"으로 활용.

### 핵심 전략
- **무료 = 광고 ON / 유료 = 광고 OFF** → "거슬리니까 돈 내고 없애자" 전환 유도
- 광고는 **거슬리지 않는 자리**에만: 통계 화면 하단, 급여 결과 화면 등
- **출퇴근 찍는 핵심 동선엔 절대 광고 금지** (사용 자체를 막으면 안 됨)
- 광고 옆에 "광고 없이 쓰기 → Basic 플랜" 버튼 배치

### 플랫폼별 도구
| 환경 | 도구 |
|---|---|
| 웹 / PWA (현재) | **Google AdSense** (사이트 심사 필요, 입금 임계 $100) |
| 안드로이드/iOS 네이티브 앱 (나중) | **Google AdMob** (SDK 삽입) |
| Apple | 자체 광고망 없음 → iOS도 AdMob 사용 |

### 시작 절차 (AdSense)
1. AdSense 가입 → crewup.kr 등록 → 사이트 심사
2. 승인 후 광고 단위 생성 → **무료 사용자 화면에만** 코드 삽입
3. 수익 $100 초과 시 등록 계좌로 입금

### ⚠️ 반드시 챙길 것
- 개인정보보호법(PIPA) — `/privacy`에 "광고 식별자 활용" 문구 추가
- **iOS ATT** — "추적 허용" 팝업 필수 (없으면 단가 급락)
- AdMob/AdSense **동의 메시지(CMP)** 설정

### 현실 수익 (참고)
- 무료 100명 × 하루 5노출 ≈ 월 3만~7만 원 수준 → **수만 명 전까지는 용돈**
- **유료 구독 1건 ≈ 무료 수백 명 광고 수익** → 구독이 메인, 광고는 보조

### 구현 준비 (미리 해둘 수 있는 것)
- [ ] `isPremium` 플래그 기반 광고 슬롯 자리를 무료 화면에 미리 배치 (나중에 코드만 꽂기)
- [ ] 개인정보처리방침에 광고/식별자 문구 추가
- [ ] "광고 없이 쓰기 → 업그레이드" 배너 컴포넌트

### 타이밍
**사용자 어느 정도 모인 뒤(2단계).** 출시 직후엔 트래픽 적어 심사도 안 붙고 수익도 0에 가까움.

## 2. 💬 크루톡 — 인증된 사장님 익명 커뮤니티

**한 줄 요약:** 블라인드처럼 동네/업종별 익명 게시판. 단, **"사장님 편"으로 설계**해야 함.

### 핵심 결정 — 절대 사장 vs 알바 섞지 말 것
- 사장님과 알바는 **이해관계 정반대** → 한 익명방에 섞으면 알바의 매장 저격 → 유료 고객(사장님) 즉시 이탈 + 명예훼손 신고가 플랫폼에 옴 (자살골)
- ✅ **"사장님 전용 동네/업종 익명 사랑방"**으로: 최저임금·주휴수당·노무 질문, 장사 노하우, 구인 정보, 세무 팁, 권리금 정보 등
- 알바 커뮤니티는 **한참 뒤 / 혹은 분리 운영** (도입 시 신중히)

### 🔑 결정적 무기 — 사업자 인증 배지
- 이미 만든 **사업자등록 진위확인**을 활용 → **"인증된 진짜 사장님만 들어오는 커뮤니티"**
- 블라인드·네이버 카페·당근에 없는 차별점 = **신뢰가 곧 해자(moat)**
- 가짜·홍보충·알바 위장 차단

### 가드레일 (필수)
- 🚫 특정 매장·실명 저격 금지 (자동 필터 + 신고)
- 🏷️ 동네·업종 카테고리로만 운영 (정보 공유 유도)
- 🛡️ 신고/숨김/차단 — 운영 안 하면 게시판 썩음

### 효과 / 수익 연결
- 리텐션 ↑ (외로운 자영업자에게 동종업계 정보 = 금쪽), 입소문 ↑
- → **구인 게시판 · 노무/세무 제휴** 매출로 자연 연결

### 타이밍
**핵심 도구 리텐션 확보 + 한 동네/업종에 사용자 밀도 생긴 뒤.** 텅 빈 게시판이 최대 적. 밀도 없으면 죽은 게시판 → 제품 후져 보임.
- (선택) 출시 전 "크루톡 출시 예정 + 동네/업종 선점 대기명단" 떡밥으로 기대감·락인 형성
