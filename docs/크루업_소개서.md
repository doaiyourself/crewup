## 1. 크루업은 왜 만들어졌나

오랫동안 동네 사장님들의 손발이 되어주던 매장 관리 앱 **‘크루몬’**이 서비스를 종료했습니다. 출퇴근·급여·스케줄을 한 곳에서 챙기던 사장님들은 다시 수기 장부와 메모, 단톡방으로 흩어진 일상으로 돌아가야 했습니다.

**‘잘 쓰던 도구가 사라졌다고, 매장 운영의 고단함까지 돌아갈 수는 없다.’** 크루업은 바로 그 빈자리에서 출발했습니다. 크루몬이 남긴 익숙함은 이어가되, 오늘의 매장에 맞게 더 단단하고 가볍게 다시 설계했습니다.

이름에는 두 가지 뜻을 담았습니다. 흩어져 있던 알바·직원(**Crew**)이 한 팀으로 정렬되고, 매장이 함께 성장하며 우상향(**Up**)한다는 의미입니다. 역할은 달라도 하나의 팀 — 그것이 크루업이 일하는 방식입니다.

> **크루몬을 쓰셨던 사장님께**
> - 익숙한 출퇴근·급여·스케줄 흐름을 그대로 이어집니다.
> - 카카오 로그인으로 1분 만에 시작하고, 별도 설치 없이 휴대폰·PC에서 바로 씁니다.
> - 매장 데이터는 매장 단위로 안전하게 분리되어 권한이 있는 사람만 열람합니다.

## 2. 크루업 한눈에 보기

크루업은 매장 운영에 필요한 일을 하나의 흐름으로 묶은 알바 관리 서비스입니다. 흩어져 있던 출근·계약·급여·할일·서류를 한 곳에 모았습니다.

| 기능 | 설명 |
|---|---|
| **출퇴근 관리** | 버튼 한 번에 출근·퇴근. 실시간 근무 현황과 지각·결근 정정까지 한눈에. GPS·QR 방식 지원. |
| **근로계약서** | 표준근로계약서 작성과 전자 서명, PDF 보관·갱신 관리를 간편하게. |
| **급여 계산** | 근무 기록 기반 자동 산정. 주휴수당·야간 가산·공제까지 명세서로 발행. |
| **스케줄 관리** | 주간 근무표를 손쉽게 편성하고, 충돌·초과근무를 미리 경고. |
| **할일 공유** | 할일을 매장에 공유하면 실행한 크루가 체크. 누가 언제 완료했는지 기록으로 남김. |
| **문서함** | 매장 서류·근로계약서·급여명세서·보건증을 한 곳에 보관하고 권한대로 열람. |
| **관리자 대시보드** | 인건비·근무시간·출근율 통계로 매장 운영을 데이터로 파악. |
| **키오스크** | 매장 태블릿을 출퇴근·할일 전용 화면으로. 기기 토큰으로 안전하게 연결. |
| **카카오 로그인** | 복잡한 가입 없이 카카오로 시작. 모바일·PC 어디서나 PWA로 바로 사용. |

## 3. 할일 공유 — 공유하고, 체크하면, 기록이 남습니다

‘오픈 청소했어요?’, ‘마감 정산 누가 했지?’ 매번 묻고 확인하던 일을 크루업이 대신합니다. 할일을 매장에 공유하면, 출근한 크루가 직접 체크하고, 누가·언제 실행했는지가 자동으로 남습니다.

### 이렇게 동작합니다
1. **등록·공유** — 사장님·점장이 오늘의 할일을 등록해 매장에 공유합니다.
2. **완료 체크** — 실행한 크루가 탭 한 번으로 완료 처리합니다.
3. **자동 기록** — 완료자 이름과 시각이 자동으로 남아 책임이 명확해집니다.

> **예: 오늘의 할일 (2 / 3 완료)**
> - ✓ 오픈 청소 — 민지 · 오전 9:02 완료
> - ✓ 재고 확인 — 준호 · 오전 10:15 완료
> - ○ 마감 정산 — 아직 미완료

매번 확인하지 않아도 매장이 알아서 굴러갑니다. 키오스크 화면에서도 동일하게 체크할 수 있어, 출근한 크루가 자연스럽게 그날의 할일을 처리합니다.

## 4. 문서함 — 매장의 서류를 한 곳에

계약서, 보건증, 사업자등록증, 임대차계약서… 매장을 운영하다 보면 챙겨야 할 서류가 많습니다. 크루업 문서함은 이 모든 파일을 한 곳에 모으고, 누가 무엇을 볼 수 있는지 권한으로 정리합니다.

### 4개의 탭으로 정리

| 탭 | 설명 |
|---|---|
| **매장서류** | 사업자등록증·임대차계약서 등 매장 운영 서류. 사장님만 등록·열람. 파일 열기/다운로드 지원. |
| **근로계약서** | 발행된 표준근로계약서. 사장님은 전 직원, 본인은 자신의 계약서를 열람. |
| **급여명세서** | 월별 급여명세서. 사장님은 전 직원, 본인은 자신의 명세서를 확인. |
| **보건증** | 본인이 직접 업로드. 본인과 사장님만 열람 가능 — 개인정보를 안전하게. |

### 권한 원칙
- **사장님** — 매장의 모든 문서를 직원별로 모아 봅니다.
- **직원·알바** — 본인의 문서(계약서·급여명세서·보건증)만 봅니다.
- **보건증** — 민감 정보라 본인과 사장님 외에는 열람할 수 없습니다.

파일은 매장 단위로 안전하게 분리 저장되며, 열람 권한이 있는 사람에게만 한시적 링크로 제공됩니다. 매장서류는 ‘열기’로 미리 보고, ‘다운로드’로 원본을 그대로 내려받을 수 있습니다.

## 5. 역할은 달라도, 한 팀입니다

권한은 매장 단위로 부여되어, 각자에게 꼭 필요한 화면만 보입니다.

| 역할 | 권한 |
|---|---|
| **사장님** | 매장 전체 관리 — 직원·급여·계약·문서·통계, 그리고 점장에게 권한 위임. |
| **점장** | 출퇴근 승인·정정, 스케줄 편성 (사장님이 위임한 범위 내). |
| **직원** | 본인 근무·계약·급여·문서 확인. |
| **알바** | 본인 출퇴근·급여·할일 확인. |

1. **crewup.kr 접속** 후 카카오로 로그인합니다.

3. **직원·알바는** 코드 또는 초대 링크로 매장에 합류합니다.
4. **홈 화면에 추가하면** 앱처럼 바로 사용할 수 있습니다 (PWA).
